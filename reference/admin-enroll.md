# 새 관리자 추가 가능

### 관리자 관리 페이지 필요
링크모음 관리, 보증업체 관리, 광고배너 관리 뿐만 아니라 관리자 관리 페이지도 추가
- sub-nav에서는 가장 왼쪽에 배치
- 관리자 관리 페이지에서는 role이 admin인 사용자 목록을 보여줌

### 관리자 추가
관리자 관리 페이지에서 새 관리자를 추가할 수 있음
- 목록 우측 상단에 추가 버튼
- 추가 버튼을 누르면 관리자 추가 모달이 뜸
- username, nickname, password를 입력하고, 생성 버튼을 누르면 생성완료

### 관리자 수정, 삭제
- 목록 row의 각 우측에 수정과 삭제 버튼 배치
- 수정을 누르면 수정 모달이 뜨고, nickname을 변경 가능
- 삭제를 누르면 "관리자를 삭제하시겠습니까?" 확인 문구 뜨고, 확인을 누르면 진짜 삭제, 취소 누르면 진행 안함

### 제한 사항
- 자기 자신은 삭제할 수 없음
- admin은 최소 하나 이상의 유저가 있어야하므로 1개 남았을 때는 삭제할 수 없음.
- 1개 남은 관리자를 삭제하려고 하면 "관리자는 최소 1명이상 있어야합니다." 문구가 toast로 뜸
